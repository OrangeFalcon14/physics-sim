How to Use the Simulator

①

Blocks

Left-click anywhere on the canvas to place a block at that location.

②

Walls

Left-click once to set the top-right corner and left-click again to set the bottom-right corner.

③

Springs

Left-click to set the starting point, then left-click again to set the maximum extension point.

④

Selecting and Adding Objects

Use the bottom left panel to choose objects like blocks, walls, or springs and add them to the canvas.

⑤

Changing Object Properties

Use the right-hand panel to adjust properties like mass, length, stiffness, and other parameters of selected objects.
